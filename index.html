Signet

The threshold signing network for social login key management.

Signet is a decentralized marketplace of signing providers that uses FROST threshold cryptography (RFC 9591) to split private keys across independent nodes. Developers get social login auth for their users with no single custodian — and a smart account (ERC-4337) that's ready from the first sign-in.
